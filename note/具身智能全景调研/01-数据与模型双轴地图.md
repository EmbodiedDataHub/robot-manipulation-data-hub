# 具身智能 · 数据与模型双轴地图

> **核心问题**：具身智能可以从「数据」或「模型」两条轴理解——它们不是对立，而是同一系统的两面。

---

## 一、一句话

```text
数据轴：demo 从哪来 → 怎么存 → 怎么对齐 → 怎么混合喂模型
模型轴：看什么、听什么 → 用什么网络 → 输出什么动作 → 怎么训/部署
```

**2026 共识**：瓶颈往往在 **数据质量与对齐**，而不是缺一个更大的 Transformer。

---

```text

    (SigLIP)     (OpenVLA)    (LoRA)       (chunk+Hz)

    ─────────────────────────┼─────────────────────────→

                    数据轴（Data-Centric）
```

## 三、数据轴：六层栈（从采集到 batch）

```text
Layer 1  数采范式     Teleop / UMI / VR / Ego / 仿真
Layer 2  原始录制     ROS bag / MCAP / HDF5 / 项目私有
Layer 3  Canonical IR 统一 episode/step、时间戳、物理语义
Layer 4  格式导出     LeRobot（微调）· RLDS（预训练/OXE）
Layer 5  混合 Recipe  策展、√N 采样、embodiment_id、负迁移诊断
Layer 6  Collator     归一化、pad/mask、chunk、tokenize → batch
```

**关键文档**：[数采到VLA训练-数据管线整体方案](../数采到VLA训练-数据管线整体方案.md) · [05-数采方法](./05-数采方法分类与详解.md) · [02-格式标准](./02-数据集格式标准与转换工具.md)

### Data Pyramid（数据分层混合）

```text
Tier 3 · 精调：目标机 Teleop/UMI（决定部署成功率）
Tier 2 · 预训练：OXE / UMI 万小时 / 仿真 bootstrap
Tier 1 · 先验：Ego 视频 / Web VLM（语义，不能单独当 motor BC）
```

## 四、模型轴：六层栈（从感知到动作）

```text
Layer 1  任务 & Benchmark   LIBERO / CALVIN / 真机成功率
Layer 2  硬件 & 控制        臂、相机、50Hz 控制环、IK/FK
Layer 3  Policy 算法        BC / ACT / Diffusion / Flow
Layer 4  Action Head        离散 token / 连续 chunk / DiT 扩散
Layer 5  VLA / FM           VLM + 大规模 robot pretrain
Layer 6  后训练（可选）      RL / DAgger / 偏好对齐
```

**关键文档**：[VLA训练范式全景图](../VLA训练范式全景图.md) · [04-VLA架构](./04-VLA模型架构与输入输出.md)

### 训练阶段（模型轴时间线）

```text
Stage 0  VLM 预训练（通常下载权重，不自己做）
Stage 1  Robot BC Pretrain（OXE / 私有 / UMI）
Stage 2  Finetune（LoRA / 全量，50–500 demo）
Stage 3  Co-train / RL（可选）
Stage 4  部署（chunk 执行 + 低频 re-infer）
```

## 六、架构路线 × 数据路线（矩阵）

| 架构路线 | 典型数据需求 | 格式偏好 |
|----------|-------------|----------|
| 纯 IL（ACT/DP） | 单任务 50–500 ep | HDF5 / LeRobot |
| Token VLA（OpenVLA） | OXE 策展 ~970K | RLDS |
| Modular（Octo） | OXE delta 子集 | RLDS |
| Flow VLA（π0） | 大规模私有 teleop | 内部 IR |
| DiT FM（RDT） | OXE + unified 128 | RLDS → HDF5/自定义 |

详见 [04-VLA模型架构与输入输出](./04-VLA模型架构与输入输出.md)

## 七、三代演进（时间轴，两轴一起看）

```text
2022  RT-1        数据：单机构 130K  |  模型：Transformer + 离散 action
2023  RT-2/OXE    数据：跨机构 1M+   |  模型：VLM + action token + web
2024  OpenVLA/Octo  开源复现 OXE   |  7B VLA / 小 modular
2024–25 π0/RDT    连续 FM           |  Flow / DiT + unified space
2025–26 RDT2/UMI scale            |  人类数据零样本迁机
```
