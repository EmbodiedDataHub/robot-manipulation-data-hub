## 二、Tier 0 · Open X-Embodiment (OXE)

| 项 | 内容 |
|----|------|
| 规模 | 1M+ episodes，22 种机器人，60+ 子集 |
| 格式 | RLDS (TFRecord) |
| 用途 | Octo / OpenVLA / RDT 等预训练主粮 |
| 元数据表 | [Google Sheet](https://docs.google.com/spreadsheets/d/1rPBD77tk60AEIGZrGSODwyyzs5FgCU9Uz3h-3_t2A9g/edit#gid=0) |

### 下载（官方 GCS）

```bash
# 安装 Google Cloud SDK 后
gsutil -m cp -r gs://gresearch/robotics/{dataset_name} ~/tensorflow_datasets/

# 示例：RT-1 子集
gsutil -m cp -r gs://gresearch/robotics/fractal20220817_data ~/tensorflow_datasets/

# 示例：Bridge
gsutil -m cp -r gs://gresearch/robotics/bridge ~/tensorflow_datasets/
```

**注意**：`dataset_name` 与元数据表一致，不要多加下划线。

### 镜像与 LeRobot 版（无 gsutil 时）

| 来源 | 链接 | 说明 |
|------|------|------|
| HF 镜像 | [jxu124/OpenX-Embodiment](https://huggingface.co/datasets/jxu124/OpenX-Embodiment) | ~1.13TB，非官方 |
| LeRobot 转换 | [openx_dataset_lerobot_v2.0](https://huggingface.co/spaces/IPEC-COMMUNITY/openx_dataset_lerobot_v2.0) | 部分子集 |
| 转换工具 | [openx2lerobot](https://github.com/Tavish9/openx2lerobot) | 自行转换 |

### 官方入口

- 官网：[robotics-transformer-x.github.io](https://robotics-transformer-x.github.io/)
- GitHub：[google-deepmind/open_x_embodiment](https://github.com/google-deepmind/open_x_embodiment)

## 三、Tier 1 · 大规模真机单数据集

### 3.1 RT-1

| 项 | 内容 |
|----|------|
| 规模 | ~130K episodes |
| 平台 | Everyday Robots 移动操作臂 |
| Action | Delta EEF + gripper (7D) |
| 格式 | OXE 子集 `fractal20220817_data` |
| 下载 | 通过 OXE `gsutil` 命令 |

### 3.2 DROID

| 项 | 内容 |
|----|------|
| 规模 | 76K ep / ~350h，564 场景，86 任务 |
| 平台 | Franka Panda |
| Action | **Absolute** EEF + gripper (7D) |
| 格式 | RLDS 1.7TB |

```bash
# RLDS 完整 v1.0.1（推荐，含完整语言标注）
gsutil -m cp -r gs://gresearch/robotics/droid/1.0.1 <your_path>

# 小样 ~2GB（本地调试）
gsutil -m cp -r gs://gresearch/robotics/droid_100 <your_path>

# 原始全分辨率 8.7TB
gsutil -m cp -r gs://gresearch/robotics/droid_raw <your_path>
```

| 资源 | 链接 |
|------|------|
| 官网 | [droid-dataset.github.io](https://droid-dataset.github.io/) |
| GitHub | [droid-dataset/droid_policy_learning](https://github.com/droid-dataset/droid_policy_learning) |
| LeRobot 版 | [cadene/droid_1.0.1](https://huggingface.co/datasets/cadene/droid_1.0.1) |

**本地样例**：`code/robot-dataset-demos` 含 `droid_100` 子集。

### 3.3 BridgeData V2

| 项 | 内容 |
|----|------|
| 规模 | ~60K demos |
| 平台 | WidowX 250 |
| Action | Delta EEF (7D) |
| 格式 | OXE 子集 `bridge` |

- OXE 下载：`gsutil -m cp -r gs://gresearch/robotics/bridge ...`
- LeRobot v3：[nvidia/bridge_lerobot_v3](https://huggingface.co/datasets/nvidia/bridge_lerobot_v3)

### 3.4 RH20T

| 项 | 内容 |
|----|------|
| 规模 | 110K+ 序列 |
| 特点 | 力/触觉/音频多模态，~5TB RGB |
| 下载 | [rh20t.github.io](https://rh20t.github.io/)（Google Drive / 百度网盘） |

## 四、Tier 2 · 平台专用 / 精细操作

### 4.1 ALOHA

| 项 | 内容 |
|----|------|
| 规模 | 任务级 50–500 ep 常够微调 |
| 平台 | 双臂 ViperX |
| Action | Joint 14D |
| 格式 | HDF5 / LeRobot |

```bash
# 仿真 LeRobot（HF，低门槛）
huggingface-cli download lerobot/aloha_sim_transfer_cube_human --repo-type dataset

# 真机项目
git clone https://github.com/tonyzhaozh/aloha
```

| 资源 | 链接 |
|------|------|
| 官方 | [github.com/tonyzhaozh/aloha](https://github.com/tonyzhaozh/aloha) |
| NVIDIA 数据集 | [nvidia/ALOHA-Cosmos-Policy](https://huggingface.co/datasets/nvidia/ALOHA-Cosmos-Policy) |

### 4.2 UMI / RDT2 人类数据

| 项 | 内容 |
|----|------|
| 规模 | 10K+ 小时 |
| 特点 | 手持夹爪 + GoPro，无真机 |
| Action | Relative EEF (7D) |
| 下载 | [RoboticsDiffusionTransformer](https://github.com/thu-ml/RoboticsDiffusionTransformer) 项目页 |

### 4.3 GELLO 等低成本 Teleop

各项目 GitHub 自带 HDF5/LeRobot；适合**自建数据**参考硬件方案。

## 五、Tier 3 · 仿真与 Benchmark

### 5.1 LIBERO（VLA 最常用 benchmark 之一）

```bash
git clone https://github.com/Lifelong-Robot-Learning/LIBERO
cd LIBERO
pip install -e .
python benchmark_scripts/download_libero_datasets.py --use-huggingface
# 指定 suite：--datasets libero_spatial
```

| 资源 | 链接 |
|------|------|
| GitHub | [Lifelong-Robot-Learning/LIBERO](https://github.com/Lifelong-Robot-Learning/LIBERO) |
| HF | [yifengzhu-hf/LIBERO-datasets](https://huggingface.co/datasets/yifengzhu-hf/LIBERO-datasets) |

### 5.2 RoboCasa

```bash
pip install robocasa
python -m robocasa.scripts.download_datasets --all
# 或：--split pretrain --source human
```

- 官网：[robocasa.ai](https://robocasa.ai)
- GitHub：[robocasa/robocasa](https://github.com/robocasa/robocasa)
- LeRobot 版：见 [any4lerobot](https://github.com/Tavish9/any4lerobot)

### 5.3 其他仿真

## 七、按目标快速选型

| 你的目标 | 优先数据集 | 体积级 |
|----------|-----------|--------|
| 理解 VLA 预训练 | OXE → fractal + bridge | 按子集 10–100GB+ |
| In-the-wild 泛化 | DROID | 1.7TB / 小样 2GB |
| 跑通微调 pipeline | LIBERO 或 `lerobot/aloha_sim_*` | 数 GB |
| 双臂精细 | ALOHA | 任务级 |
| OpenVLA 复现 | OXE 策展 ~970K（见 OpenVLA 配置） | 数百 GB 级 |
| RDT 预训练 | OXE 46 集 | ~21TB 级（可子集） |
| 仿真 bootstrap | RoboCasa / MimicGen | 数十–数百 GB |

## 八、下载环境与技巧

### 8.1 依赖

```bash
# GCS
pip install gsutil  # 或安装 Google Cloud SDK

# Hugging Face
pip install huggingface_hub
export HF_ENDPOINT=https://hf-mirror.com   # 国内可选镜像
```

### 8.2 robot-dataset-demos 一键小样

```bash
cd code/robot-dataset-demos
chmod +x scripts/download_samples.sh
./scripts/download_samples.sh --lerobot-only   # ~67MB
./scripts/download_samples.sh --all            # + LIBERO + droid_100 ~2GB
```

### 8.3 体量参考

| 数据集 | 体积 | 难度 |
|--------|------|------|
| OXE 全量 | 1TB+ | 中（可只下子集） |
| DROID RLDS | 1.7TB | 高 |
| droid_100 | ~2GB | 低 |
| LIBERO 全 suite | 数 GB | 低 |
| RH20T RGB | ~5TB | 高 |

## 九、Action 空间混训警示

| 数据集 | Action 类型 | 混训注意 |
|--------|------------|----------|
| RT-1 / Bridge | Delta EEF 7D | OpenVLA 默认 |
| DROID | **Absolute** EEF 7D | 与 Bridge 冲突，需转换或分 head |
| ALOHA | Joint 14D | 需 IK 或 joint head |
| UMI | Relative EEF 7D | RDT2 路线 |

详见：[预训练数据集清单 §1.2–1.3](../paper-note/Data-Pipeline/预训练数据集清单与格式对齐.md)
