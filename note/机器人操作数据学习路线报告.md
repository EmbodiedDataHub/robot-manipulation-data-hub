```

│                        Layer 4: Foundation Models                        │

┌──────────────────────┬──────────────────────┬───────────────────────────┐
```

## 二、四条主线的关系与演进

| 维度 | ALOHA | UMI | Open-TeleVision | RDT |
|------|-------|-----|-----------------|-----|
| **核心贡献** | 低成本双臂硬件 + ACT | 手持夹爪野外采数据 | VR 主动立体视觉遥操作 | Diffusion/VLM Foundation Model |
| **采集是否需要机器人** | ✅ 需要 | ❌ 不需要 | ✅ 需要 | 消费上述数据 |
| **机器人类型** | 双臂夹爪 (ViperX) | 任意夹爪平台 | 人形 (H1, GR-1) | 跨 embodiment |
| **感知方式** | 固定腕部/顶部相机 | 腕部 GoPro Fisheye | 主动立体头显相机 | 多相机 + 语言 |
| **策略算法** | ACT (CVAE + Chunking) | Diffusion Policy | ACT | DiT / 7B VLM + Flow Matching |
| **数据规模** | 小（实验室，分钟级） | 中（多环境，小时级） | 中（长时序精细任务） | 大（1M~10k+ 小时） |
| **泛化能力** | 任务内 | OOD 环境/物体 ~70% | 跨地域远程 | Zero-shot 跨物体/场景/指令/embodiment |
| **成本** | ~$20k | ~$数百（手持设备） | VR + 人形机器人 | 训练算力 |

## 五、逐篇论文重点总结

---

### A. 核心主线（4 篇）⭐ 必读

#### 1. ALOHA — Learning Fine-Grained Bimanual Manipulation with Low-Cost Hardware

- **机构**：Stanford / UC Berkeley / Meta
- **问题**：精细操作（插电池、开 condiment cup）通常需要昂贵机器人和传感器
- **方法**：
  - 硬件：两套 ViperX 300 臂（Leader + Follower），3D 打印 backdriving 件，总成本 < $20k
  - 算法：**ACT (Action Chunking with Transformers)** — CVAE 预测 k 步 action chunk + temporal ensembling
- **结果**：10 分钟 demo → 80–90% 成功率（开 cup、插电池）
- **意义**：奠定低成本双臂遥操作 + imitation learning 范式，后续 Mobile ALOHA、RDT 均以此为 baseline robot

#### 2. UMI — Universal Manipulation Interface

- **机构**：Stanford / Columbia / TRI
- **问题**：Teleop 贵且受环境限制；纯 human video embodiment gap 大
- **方法**（六大接口设计）：
  1. 腕部 GoPro + 155° Fisheye
  2. 侧面镜子（隐式 stereo）
  3. IMU-aware pose tracking
  4. 连续夹爪追踪
  5. **Inference-time latency matching**
  6. **Relative trajectory action** + Diffusion Policy
- **结果**：动态/双臂/精确/长时序任务；OOD 测试 ~70% success
- **意义**：「无需机器人采数据」范式，RDT2 的 10k 小时数据即基于此

#### 3. Open-TeleVision — Teleoperation with Immersive Active Visual Feedback

- **机构**：UC San Diego / MIT
- **问题**：第三人称/静态相机遥操作遮挡严重，缺乏空间理解
- **方法**：
  - **主动立体头显相机**（2–3 DoF gimbal 跟随操作者头部）
  - VR 流式 stereo 视频到 Apple Vision Pro
  - Hand/head/wrist pose → IK + motion retargeting → 人形机器人
  - 策略：ACT + stereo input
- **平台**：Unitree H1（6 DoF 手）+ Fourier GR-1（jaw gripper）
- **结果**：长时序精细任务（can sorting, insertion, folding）；Boston↔San Diego 远程 60Hz
- **意义**：人形机器人沉浸式遥操作标杆，感知创新 > 执行创新

#### 4. RDT-1B — A Diffusion Foundation Model for Bimanual Manipulation

- **机构**：清华大学
- **问题**：双臂数据稀缺 + 多模态 action 分布 + 多机器人 heterogeneity
- **方法**：
  - **Robotics Diffusion Transformer (DiT)**：1.2B 参数
  - **Physically Interpretable Unified Action Space**
  - Pre-train：46 数据集 1M episodes → Fine-tune：6K+ 自采双臂数据
- **结果**：比 baseline 成功率 +56%；zero-shot / 1–5 shot / 语言指令
- **意义**：首个大规模 diffusion-based 双臂 manipulation FM
- **视频入门**：[Bilibili：RDT-1B 解密…](https://www.bilibili.com/video/BV1FjyHYmEDQ)（机器之心 × 刘松铭，~76 min）· [解读笔记](./paper-note/RDT-Foundation-Models.md)

#### 5. RDT2 — Exploring the Scaling Limit of UMI Data

- **机构**：清华大学
- **问题**：VLA 数据稀缺、架构效率低、无法 cross-embodiment
- **方法**：
  - 数据：**10,000+ 小时**增强版 UMI（100 设备 × 100+ 家庭）
  - 架构：7B Qwen2.5-VL + 三阶段训练（RVQ → Flow Matching → 单步蒸馏）
- **结果**：首个同时 zero-shot 泛化 **物体+场景+指令+embodiment** 四因素
- **意义**：UMI 数据规模化的天花板探索，Data Pyramid 理论框架

### B. Foundation Models（根目录，2 篇）

#### 6. Qwen-RobotManip Technical Report

- **机构**：Qwen Team（阿里）
- **核心**：Alignment-first — 表示/运动/行为三维对齐
- **数据**：~38,100 小时（开源机器人数据 + 人类 ego 视频 synthesis，15 平台）
- **方法**：Camera-centric delta pose + in-context policy adaptation + VLM co-training
- **结果**：OOD benchmark SOTA（RoboCasa365, LIBERO-Plus, EBench 等）
- **与 RDT 对比**：同为 FM，Qwen 强调 alignment 公式；RDT 强调 diffusion + UMI 数据 scale

#### 7. ABot-M0

- **机构**：AMAP CV Lab（阿里）
- **核心**：「One-brain, many-forms」— 6 个公开数据集清洗 → **UniACT-dataset**（600 万 trajectories, 9500 小时）
- **方法**：异构 raw data → 统一高效表示；联合优化架构与训练策略
- **意义**：数据 curation pipeline 参考，与 Qwen-RobotManip 互补

### C. Robot Teleoperation（7 篇）

#### 8. ALOHA 2 — An Enhanced Low-Cost Hardware for Bimanual Teleoperation

- **机构**：Google DeepMind + Stanford
- **升级**：更强夹爪、更高鲁棒性、更大规模 demo 数据集
- **意义**：ALOHA 官方续作，工业级硬件迭代

#### 9. Mobile ALOHA — Learning Bimanual Mobile Manipulation

- **机构**：Stanford
- **硬件**：ALOHA + 移动底座（$32k），whole-body tethered teleop
- **学习**：16D action（14 arm + 2 base velocity）；**co-training 静态 ALOHA 数据 → +34% 成功率**
- **任务**：做饭、开柜、按电梯、推椅子等

#### 10. GELLO — A General, Low-Cost Teleoperation Framework

- **核心**：通用、低成本、直观遥操作框架，可适配多种机械臂
- **意义**：ALOHA 的「通用版」，降低不同 robot 的 teleop 门槛

#### 11. AirExo — Low-Cost Exoskeletons for Learning Whole-Arm Manipulation in the Wild

- **核心**：低成本双臂外骨骼 → teleop + **野外 in-the-wild 演示采集**
- **方法**：3 分钟 teleop demo + 大量野外数据 → 机器人 whole-arm manipulation
- **意义**：外骨骼 + 野外数据的 early exploration

#### 12. Giving Robots a Hand — Eye-in-Hand Human Video Demonstrations

- **核心**：用**大量人类 eye-in-hand 视频**增强少量机器人 demo
- **方法**：跨 visual domain gap，提升 eye-in-hand visuomotor policy 泛化
- **意义**：Human video augmentation 思路

#### 13. TidyBot++ — An Open-Source Holonomic Mobile Manipulator

- **核心**：开源、低成本、**全向移动**操作平台（holonomic base）
- **接口**：手机 teleop，支持任意 arm 组合
- **意义**：Mobile manipulation 硬件参考

#### 14. Dobb-E — On Bringing Robots Home

- **核心**：家用 generalist robot；**「The Stick」** 演示采集工具（iPhone + 廉价部件）
- **数据**：13 小时 home 数据；5 分钟 demo 学新任务
- **意义**：Consumer robotics 数据采集 early work

### D. Hand-Held Grippers / UMI 生态（8 篇）

#### 15. FastUMI — Scalable and Hardware-Independent UMI

- **机构**：上海 AI Lab 等
- **改进**：
  - 硬件解耦：RealSense T265 替代 GoPro VIO
  - ISO 标准相机 mount，适配 Franka/Flexiv/Z1 等
  - 10,000+ trajectories, 22 任务
- **意义**：UMI 工程化最佳实践

#### 16. DexUMI — Using Human Hand as the Universal Manipulation Interface

- **核心**：**人手作为自然接口** → 灵巧手技能迁移
- **硬件**：可穿戴外骨骼（haptic + kinematic adaptation）
- **软件**：Human hand inpainting 替换为 robot hand
- **结果**：两种 dexterous hand 平台，平均 80%+ success

#### 17. DexWild — In-the-Wild Dexterous Manipulation

- **核心**：多人用手直接采集 dexterous manipulation 数据（无需 robot）
- **意义**：UMI 思路延伸到 dexterous / 纯人手

#### 18. Visual Imitation Made Easy

- **核心**：商用 reacher-grabber 工具作数据采集 + 机器人 end-effector
- **方法**：SfM + finger detection 提取 action
- **意义**：UMI 前身思路

#### 19. Grasping in the Wild — 6DoF Closed-Loop Grasping

- **核心**：低成本 handheld 接口采集 6DoF 抓取 demo
- **方法**：Action-view rendering + RL → 真实机器人闭环 6DoF 抓取
- **意义**：Early closed-loop grasping from human demo

#### 20. CDF-Glove — Cable-Driven Force Feedback Glove

- **核心**：轻量低成本力反馈手套（20 DoF，16 直接 sensing）
- **意义**：Dexterous teleop 触觉反馈

### F. Ego Human Data（13 篇）

#### 23. EgoMimic — Scaling Imitation Learning via Egocentric Video

- **硬件**：Project Aria 眼镜 + 低成本双臂机器人

#### 24. EgoVLA — Learning VLA Models from Egocentric Human Videos

- **方法**：Human video 训练 VLA → IK/retargeting → 少量 robot demo fine-tune
- **Benchmark**：Ego Humanoid Manipulation Benchmark

#### 25. EgoBridge — Domain Adaptation for Generalizable Imitation from Egocentric Human Data

- **方法**：Optimal Transport 对齐 human/robot policy latent space
- **结果**：比 cross-embodiment baseline **+44%** success

#### 26. EgoScale — Scaling Human Video to Dexterous Manipulation

- **数据**：**20,854 小时** action-labeled egocentric video（20× 先前工作）
- **发现**：Log-linear scaling law；validation loss 与真实机器人性能强相关

#### 27. EgoZero — Training Robots Without Robots

- **硬件**：Project Aria smart glasses
- **核心**：**零 robot 数据** — 纯 human demo → morphology-agnostic state → 闭环 policy
- **部署**：Franka Panda gripper

#### 28. Phantom — Training Robots Without Robots Using Only Human Video

- **核心**：Human video → imitation policy → **zero-shot robot deploy**（无 robot 采集）

#### 29. EgoHumanoid — Unlocking In-the-Wild Humanoid Loco-Manipulation

- **核心**：首个 ego human data → humanoid loco-manipulation VLA co-training
- **方法**：View alignment + action alignment pipeline

#### 30. EgoMI — Learning Active Vision and Whole-Body Manipulation

- **核心**：人类主动视觉（head-hand coordination）→ 缩小 embodiment gap

#### 31. EMMA — Egocentric Mobile Manipulation

- **核心**：Human full-body motion + static robot data co-train → mobile manipulation（无需 mobile teleop）

#### 32. Humanoid Policy ∼ Human Policy (HAT)

- **数据**：PH2D egocentric task-oriented dataset
- **方法**：Human Action Transformer — 统一 human/humanoid state-action space

#### 33. MotionTrans — Human VR Data Enable Motion-Level Learning

- **方法**：30 human-robot tasks co-training → **13 任务 zero-shot 迁移**，9 任务非零成功率

#### 34. Emergence of Human to Robot Transfer in VLA Models

- **核心**：研究 VLA 何时 emergent 出现 human→robot transfer 能力
- **意义**：理论性分析，指导数据混合策略

### H. Video Teleoperation（9 篇）

#### 39. MimicGen — Scalable Robot Learning using Human Demonstrations

- **核心**：**~200 human demo → 自动合成 50K+ demo**（18 任务）
- **意义**：数据扩增系统，与 UMI/ALOHA 采集互补

#### 40. VideoDex — Learning Dexterity from Internet Videos

- **方法**：从 human video 提取 visual + action + physical priors → 引导 robot dexterity
- **意义**：Internet video → robot dexterity early work

#### 41. Robotic Telekinesis — Learning by Watching Humans on YouTube

- **核心**：单 RGB 相机观察人类 → 实时 imitation 到 robot hand-arm
- **数据**：Internet hand video corpus

#### 42. Affordances from Human Videos as a Versatile Representation

- **核心**：Human video → affordance 表示 → robot 直接可用
- **意义**：Intermediate representation 思路

#### 43. AnyTeleop — General Vision-Based Dexterous Teleoperation

- **核心**：**统一** vision-based teleop，跨 robot model / 仿真环境
- **结果**：比 task-specific 系统更高 success rate
