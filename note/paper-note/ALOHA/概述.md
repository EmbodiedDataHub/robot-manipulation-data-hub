## 一句话

ALOHA 提供 **<$20k 双臂 teleop 采数据**；ACT 用 **CVAE + Transformer** 一次预测 **100 步 action chunk**，在约 **10 分钟 demo** 上完成毫米级双臂操作（**80–90%** 成功率）。CNNMLP 是论文里的 **逐步 MSE baseline**。

## 核心概念速查

| 概念 | 含义 |
|------|------|
| **qpos** | 14 维关节角 + 夹爪（双臂 absolute joint target） |
| **chunk** | 一次预测的 future 100 步动作 |
| **query** | 调用一次 `policy()` 获取新 chunk |
| **CVAE Encoder** | 训练时：demo 动作 → z；推理时关闭 |
| **z=0** | 推理时 latent 用先验均值 |
| **CNNMLP** | 每步只预测 1 步，MSE baseline |
