## 一句话

机器人 VLA 的数据工作 = **五条采集范式** → **Canonical Episode IR** → **QA + 双轨导出(RLDS/LeRobot)** → **模型 Collator** → 训练。模型可以换，IR 不能省。

## 格式标准速查

| 格式 | 用途 |

| **MCAP** | ROS2 原始录制 |
