## 四、关键公开数据集（数据工作常接触）

| 数据集 | 规模 | 特点 | 格式 |
|--------|------|------|------|
| Open X-Embodiment | 1M+ episodes, 22 robots | 跨机构聚合预训练 | RLDS |
| DROID | 76k episodes | in-the-wild, 多场景光照 | RLDS |
| BridgeData V2 | ~60k demos | 低成本 WidowX 标准 setup | RLDS |
| RT-1 / RT-2 | fleet-scale | 单 embodiment 高质量 | 私有 + 部分开源 |
| RH20T | 多模态 | 力 / 触觉 / 音频 | 自定义 |
| Ego4D / EgoScale | 万~万小时级 | 人类 ego 视频 | 视频 + 标注 |
| RDT2 UMI Data | 10k+ 小时 | 手持野外大规模 | 项目自有 |
| UniACT (ABot-M0) | 600 万 trajectories | 清洗统一后的混合数据 | 项目自有 |
| MimicGen 合成 | 50K+ from 200 seed | 仿真数据扩增 | RoboSuite |

## 五、数据格式速查

| 格式 | 适用场景 | 工具链 |
|------|----------|--------|
| **RLDS / TFRecord** | 跨 embodiment 大规模预训练 (OXE) | TFDS, Octo, OpenVLA |
| **LeRobot (Parquet + MP4)** | Hugging Face 生态、PyTorch 训练 | lerobot-train, SmolVLA, ACT |
| **HDF5** | 本地高性能、ACT 等传统 IL | robomimic, ACT 官方 |
| **MCAP** | ROS2 实时录制、索引随机访问 | rosbag2 |
| **Zarr** | 云原生分块、并行写入 | 大规模分布式训练 |
| **Robo-DM (EBML)** | 极致压缩 + 高速解码 | fog_x toolkit |
