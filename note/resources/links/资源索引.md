## 怎么用本索引

```
入门（10–30 min）  →  视频 / 项目页 demo / 中文博客
        ↓
建立结构（1–2 h）  →  本地 paper-note 笔记
        ↓
精读原文（按需）    →  本地 PDF（优先 .zh.mono.pdf）
        ↓
动手 / 部署        →  GitHub · LeRobot · HuggingFace
```

| 资料类型 | 适合什么时候看 | 本仓库入口 |
|----------|----------------|------------|
| **视频 / 演讲** | 第一次接触某篇工作；通勤背景听 | 下文 §二 |
| **博客 / 教程** | 补工程视角、数据管线、行业观点 | 下文 §三 |
